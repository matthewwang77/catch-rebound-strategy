## 八、关键发现总结

### ✅ 确认有效的规律
1. **连板数 ≥3 是关键**：从2提升到3，信号质量飞跃（胜率 +10pp，Sharpe +0.43）
2. **回调深度下限重要**：至少回调12%才入场，过滤弱回踩
3. **止损位最敏感**：-11.2% 附近是最优区间，偏差1%就会显著影响表现
4. **缩量不是越严越好**：volume_shrink=0.67（可放大到67%的涨停量）优于更严的0.3-0.5

### ❌ 确认无效的规律（辛普森悖论再次验证）
1. **超跌反弹过滤无效**：在任何网格搜索中都被淘汰
2. **低位收盘过滤无效**：边际改善，但减少信号过多
3. **pullback_ratio_max 几乎无关**：敏感度 0.00，只要回调够深，更深无所谓

### ⚠️ 风险提示
1. **熊市保护不足**：2023年熊市中两个版本都亏损（Sharpe < -2）
2. **信号少**：v5 在16个月内仅79个信号（约5个/月），实盘可能不够
3. **样本外时间短**：主要测试仅16个月，需持续跟踪

## 九、推荐使用方案

### 双模式设计

> **LOOSE 是 STRICT 的严格超集**：所有满足 STRICT 条件的股票，一定也满足 LOOSE。差异仅在连板数（3→2）和实体板比例（0.55→0.3）。两个模式的 `pullback_max` 和 `volume_shrink` 已统一，不会出现 STRICT 有但 LOOSE 没有的股票。

| 特性 | STRICT（严格） | LOOSE（宽松） |
|------|---------------|---------------|
| 适用场景 | 震荡市 / 方向不明 | 牛市 / 强趋势 |
| 核心逻辑 | 3连板 → 精选高质量 | 2连板 → 广撒网（STRICT的超集） |
| 信号频率 | ~5笔/月 | ~18笔/月 |
| 胜率 | 69.6% | 61.1% |
| Sharpe | 1.71 | 1.54 |
| MaxDD | -28.8% | -39.9% |

### STRICT 模式（默认）

```python
# 震荡市/不明市首选 — 79信号/16月, 69.6%胜率, Sharpe 1.71
PARAMS = {
    "min_consecutive_limit_up": 3,    # 核心：需3连板
    "min_entity_board_ratio": 0.55,
    "pullback_ratio_min": 0.12,       # 回调至少12%
    "pullback_ratio_max": 0.40,
    "volume_shrink_ratio": 0.67,
    "volume_shrink_ratio_min": 0.05,
    "take_profit": 0.051,             # 止盈+5.1%
    "stop_loss": -0.112,              # 止损-11.2%
    "hold_days": 10,
}
```

### LOOSE 模式（牛市）

```python
# 牛市/强趋势 — 285信号/16月, 61.1%胜率, Sharpe 1.54
# 是STRICT的严格超集（pullback_max和volume_shrink一致）
PARAMS = {
    "min_consecutive_limit_up": 2,    # 核心：2连板即可（vs STRICT的3）
    "min_entity_board_ratio": 0.3,    # （vs STRICT的0.55）
    "pullback_ratio_min": 0.08,       # （vs STRICT的0.12）
    "pullback_ratio_max": 0.40,       # = STRICT
    "volume_shrink_ratio": 0.67,      # = STRICT
    "volume_shrink_ratio_min": 0.0,
    "take_profit": 0.05,
    "stop_loss": -0.10,
    "hold_days": 7,
}
```

### 双模式跨周期验证

| 时段 | 市场 | STRICT (信号/WR/Sharpe) | LOOSE (信号/WR/Sharpe) | 推荐 |
|------|------|------------------------|------------------------|------|
| 2023.01-2024.06 | 熊市🐻 | 38/42%/-2.08 | 107/39%/-2.32 | 都不开 |
| 2024.07-2025.06 | 牛市🐂 | 90/69%/0.87 | **354/74%/2.15** | LOOSE |
| 2025.01-2026.04 | 震荡📊 | 79/70%/1.71 | 285/61%/1.54 | STRICT(质) LOOSE(量) |

### 使用方式

```bash
# 当日选股 — 严格模式（默认）
python 选股new_v5.py --today strict

# 当日选股 — 宽松模式（牛市用）
python 选股new_v5.py --today loose
```

### 切换建议

- **默认使用 STRICT**：信号质量高、回撤可控
- **检测到牛市 → 切 LOOSE**：大盘在20日均线上方 + 成交量放大 + 涨停家数增多
- **检测到熊市 → 暂停**：大盘在20日均线下方且下跌趋势中，两个模式都不开仓
