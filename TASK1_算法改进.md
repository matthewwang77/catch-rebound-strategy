# 任务1：改进抓反弹策略算法，提高胜率

请基于华安证券研报（32,615 个首板样本分析）的研究成果，改进我的连板回调策略。

## 研究参考

### 五大反直觉规律
1. **缩量远优于放量**：量比<0.3x 的样本胜率 91.7%、均收益 +7.44%；量比>1.0x 反而亏损 -0.71%
2. **非多头排列更优**：非多头排列首板次日 +0.40%，优于多头排列的 +0.22%
3. **超跌首板强 10 倍**：前 20 日跌 >10% 的首板均收益 +1.04%，高位首板仅 +0.10%
4. **市值 U 型分布**：极小市值（<15 亿）和大市值（>100 亿）优于中等市值
5. **低位收盘更佳**：回调日收盘在日内低位的标的次日表现显著更好

### 最有效的优化方向（来自研报回测）
- 收紧回调上限至 +3% → **+18.3% 收益提升**（单项最大）
- 移除换手率上限 → +6.3%
- 过滤高位收盘回调 → +3.9%
- 阶段自适应止损 → +3.2%

### ⚠️ 辛普森悖论
单变量分析中表现优异的因子，进入多因子策略后可能逆转失效。所有改动需通过完整回测验证。

## 需要改动的文件

### 1. `选股new.py` — 核心改动

**A. 收紧 strict 模式参数**
- pullback_ratio_max: 0.25 → 0.15
- volume_shrink_ratio: 0.4 → 0.3
- 新增 PARAM: volume_shrink_ratio_min: 0.05（极端缩量更好）

**B. 新增超跌反弹过滤器**
- 新增 PARAM: require_oversold (bool), oversold_decline_threshold: 0.10
- 在 _screen_single_stock 中：检查涨停前 20 日跌幅是否 >10%

**C. 新增低位收盘过滤器**
- 新增 PARAM: require_low_close (bool)
- 信号日收盘价应在当日振幅下半区（close_position < 0.5）

**D. 新增 "oversold" 第五种筛选模式**
- 聚焦超跌反弹：回调 15%-30%、严格缩量<0.3x、要求超跌、不要求阳线放量

**E. 更新优化评分函数**
- 当前：win_rate*0.3 + total_return*0.5 + count*0.2
- 改为：win_rate*0.5 + total_return*0.35 + count*0.15

### 2. `streamlit_app.py` — 展示改动

**F. 显示市场情绪阶段**
- 在 `真实版V2.py` 中已有滚动情绪计算逻辑，参照实现
- 大盘概况下方显示：当前情绪阶段（高潮期/衰退期/冰点期）

**G. 新增 "oversold" Tab**
- 四种模式 Tab → 五种

### 3. 同时更新 `真实版V2.py` 的评分函数
- win_rate*0.4 + avg_ret*3 + count*0.3 → win_rate*0.55 + avg_ret*2.5 + count*0.2

## 验证方式
- 运行 `python 选股new.py --optimize` 对比优化前后结果
- 运行 `python 选股new.py` 完整回测，对比胜率、均收益、夏普比率
- 确保新参数不会大幅减少信号数量（信号太少无法交易）
